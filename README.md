# ⚡ FlashChat.ai

**FlashChat.ai** is a modern AI-powered chatbot built with **React**, **FastAPI**, and **OpenAI GPT** that delivers fast, intelligent, and natural conversations. It features a clean, responsive interface and a scalable architecture, making it an excellent project for learning full-stack AI development and showcasing in a professional portfolio.

## ✨ Features

* 🤖 AI-powered real-time conversations
* 💬 Context-aware chat history
* ⚡ Fast and responsive user interface
* 🎨 Modern React + Vite frontend
* 🚀 FastAPI backend with REST APIs
* 🔐 User authentication (coming soon)
* 💾 Chat history storage with SQLite
* 🌙 Dark mode support
* 📱 Fully responsive design
* 🔌 Easy integration with multiple AI providers

## 🛠️ Tech Stack

FlashChat.ai aims to provide a fast, intuitive, and intelligent conversational AI experience. The project is designed with scalability in mind, allowing future integration of advanced AI features while maintaining a clean and maintainable codebase.
